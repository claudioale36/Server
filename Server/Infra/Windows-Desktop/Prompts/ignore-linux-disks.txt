- He realizado una instalacion limpia de Windows 11, y nuestro objetivo es evitar que esto vuelva a ocurrir. Para lograr esto, ya he configurado Linux para que ignore COMPLETAMENTE el disco Windows, incluso que Grub no lo escanee.
